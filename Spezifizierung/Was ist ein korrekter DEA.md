# DEA

Ein korrekter DEA richtig sich an die Mathematische vorlage. Das bedeutet er hat 5 bestimmte Parameter.

1. nicht leere endliche Menge von Zuständen
2. nicht leere endliche Menge von Zeichen (Alphabet)
3. determinitsich vollständige Menge von Zustandübergänge
4. ein nicht leere unter Menge von Zustände (können auch alle Zustände sein), welche die Endzustände repräsentieren
5. genau einen Zustand aus der Menge der Zustände, der als Starzustand agiert.

**Anmerkung**:
vollständige Menge vno Zustandsübergänge bedeutet im klaren, das es von jedem Zustand einen Nachfolgende Zustand gibt für jedes Zeichen vom Alphabet. Das deterministisch heißt, das es von einen Zustand keine zwei oder mehr Nachfolgenden Zustände geben kann für ein Zeichen.

Allgemein Anmerkung:
Menge bedeutet es gibt keinen dopplungen.

Wenn einer dieser Punkte nicht erfüllt wird, gilt der DEA, laut dieser Spezifierzierung nicht als korrekt.
